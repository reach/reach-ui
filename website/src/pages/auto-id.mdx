# Auto ID

Autogenerate IDs to facilitate WAI-ARIA and server rendering.

## Installation

```bash
npm install @reach/auto-id
# or
yarn add @reach/auto-id
```

## Usage

```js
import { useId } from "@reach/auto-id";

const id = `tooltip:${useId()}`;
```
